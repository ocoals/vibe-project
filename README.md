## 저장소 구성

|------|------|
| [`frontend/`](frontend/) | Next.js 앱 (App Router, TypeScript, Tailwind, Supabase) |
| [`docs/`](docs/) | 제품·API 명세 등 ([`spec.md`](docs/spec.md), [`api-spec.md`](docs/api-spec.md)) |
| [`supabase/`](supabase/) | Supabase 설정·마이그레이션 |
